使用nvm进行node版本管理

发表于 2017-02-28

优雅的使用nvm能够快速无缝切换node版本

nvm是什么

nvm

nvm全称Node Version Manager，是Node的版本管理器，类似于Ruby的rvm。由于node的迭代速度快，并且版本多，所以运用版本管理器对于Node开发是非常有必要的。

作用

使用nvm能够使Node的各个版本分离，存储在nvm的目录下,能够快速切换Node版本，并且不同版本之间的环境不会互相影响。

安装

卸载全局的Node

全局安装的Node安装在/usr/local/lib/node_modules/npm中，命令在/usr/local/bin/node

下载nvm

使用

下载各个版本的node，stable是最新的稳定版，下载完之后可以开始使用nvm转化管理和切换版本了。

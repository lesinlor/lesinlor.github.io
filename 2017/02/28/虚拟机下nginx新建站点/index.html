虚拟机下nginx新建站点

发表于 2017-02-28

一开始在同事的帮助下成功安装了虚拟机，现在平日做demo也在虚拟机环境下运行，所以需要新建多个端口，也方便以后多个项目跑起来。

找到nginx的目录

$ whereis nginx

进入nginx目录下

命令行输入进入目录$ cd sites-available，此目录存放每一个端口的配置。

default是默认端口，复制一个文件$ sudo cp default test，进行更改$ sudo vim test

之后要在sites-enabled文件夹下建立软连接$ ln -s /etc/nginx/sites-available/beta /etc/nginx/sites-enabled/前一个路径为文件的名字，后一个路径为要创建软连接的路径，必须都为绝对路径。

更改完两处之后端口设置完成。

新建web

在本机或者虚拟机的共享目录下新建对应root的文件夹，并在文件夹下新建index便可以开始访问新的端口

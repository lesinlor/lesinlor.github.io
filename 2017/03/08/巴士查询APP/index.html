巴士查询APP

发表于 2017-03-08

经过vue学习之后，简单运用过vue的双向绑定，然而这只是vue最基础的东西，vue能做的远远不止这些，所以我准备运用起vue全家桶尝试开发一个简单的spa项目

vue全家桶

  • vue
  • vue-router
  • vuex
  • vue-cli

环境

  • 系统:Nginx Ubuntu 14.04
  • Node.js v7.2.1
  • vue: 2.1.10
  • vue-router 2.2.0”
  • vuex 2.1.2”

项目起步

项目的架构采用vue-cli进行构建，先进行vue-cli的安装或升级
$ npm install/update vue-cli
进入项目的目录开始初始化项目
$ vue init webpack bus-server。
进入项目安装依赖并启动dev脚本:
$ npm i
npm run dev

启动dev脚本后，webpack dev middleware的检测并不能成功检测，并刷新页面。后来在Stack Overflow 找到了答案。（主要是修改了dev-server.js的配置，具体位置不太记得了）

然后写入依赖的包
$npm install vuex vue-router --save

项目结构

|-- App.vue
|-- as.html
|-- assets
|   |-- logo.png
|-- components
|-- Home.vue
|-- main.js
|-- models
|   |-- api.js
|-- router
|   |-- router.js
|-- store
|   |-- index.js
|-- utils
|   |-- utils.js
|-- views
|   |-- Busquery.vue
|   |-- Bus.vue
|   |-- Stationquery.vue
|   |-- Station.vue

router存放路由信息，view存放页面，store存放vuex状态管理的js，utils存放公共的js，这里写了一个原声的ajax请求发送。

接口

接口运用的是聚合的API，是免费申请的接口，处理方法。

项目地址
